# Copilot Instructions

## Propósito
Este proyecto es un sistema básico de ecommerce construido con Java y Spring Boot. El objetivo principal es aprender Java y desarrollo web con Spring Boot mientras se desarrolla una aplicación práctica.

## Alcance del sistema
- Backend en Java 17
- API REST con Spring Boot
- Funcionalidades básicas de ecommerce como gestión de usuarios y productos
- Uso de H2 para desarrollo y PostgreSQL en producción

## Requisitos generales
- Mantener el código limpio, claro y modular
- Usar convenciones estándar de Spring Boot y Java
- Priorizar la educación y la legibilidad sobre soluciones demasiado complejas

## Estilo de respuesta
- Responder en español
- Ser breve y directo
- Explicar los cambios realizados y el motivo cuando corresponda
- Proveer rutas, archivos y ejemplos concretos si se agregan endpoints o tests
- Aunque las instrucciones y el lenguaje del chat son en español, todo el código debe escribirse en inglés

## Notas
- Este sistema debe servir como entorno para aprender Java, Spring Boot y conceptos básicos de ecommerce
- Evitar soluciones avanzadas que no aporten valor educativo inmediato
